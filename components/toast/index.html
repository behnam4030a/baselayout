2. بدون Description

Toast فقط با عنوان — بدون فضای خالی اضافی.

عملیات با موفقیت انجام شد

3. دمو تعاملی (Interactive)

روی دکمه‌ها کلیک کنید — Toast در گوشه بالا-چپ ظاهر می‌شود و بعد از 5 ثانیه خودکار حذف می‌شود.

کد نمونه:

Toast.show({
  variant: 'success',
  title: 'عملیات با موفقیت انجام شد',
  description: 'تغییرات شما ذخیره شد.',
  duration: 3000,          // اختیاری (پیش‌فرض: 5000ms)
  position: 'top-left'     // اختیاری (پیش‌فرض: 'top-left')
});
