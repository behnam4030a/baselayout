۸. CSS Classes

کلاس توضیح
switchکلاس پایه (روی <label> wrapper)
switch__inputinput بومی مخفی (<input type="checkbox" role="switch">)
switch__trackنوار پس‌زمینه (40px × 20px)
switch__thumbدایره متحرک (22px × 22px)
switch__iconآیکن داخل Thumb (14px)
switch__icon--checkآیکن checkmark (حالت checked)
switch__icon--closeآیکن X (حالت unchecked)
switch--disabledحالت غیرفعال (fallback برای :has())
