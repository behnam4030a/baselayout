Typography Component

کامپوننت تایپوگرافی - Pure CSS Component

درباره این کامپوننت:

تایپوگرافی تو UI یعنی «سیستم کنترل متن»؛ نه فقط انتخاب فونت.

تایپوگرافی شامل چیه؟

  • Family name — اسم خانواده‌ی فونت: Peyda(FaNum)
  • Weight — ضخامت حروف: Regular, Medium, SemiBold, Bold
  • Size — اندازه‌ی متن: از 12px تا 58px
  • Line height — فاصله‌ی عمودی بین خط‌ها

این کامپوننت 22 استایل متنی ارائه می‌دهد و فقط CSS است (بدون JavaScript).
