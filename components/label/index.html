Label Component

برچسب‌ها به یک کامپوننت یا گروهی از کامپوننت‌ها نام می‌دهند. آن‌ها را با اجزایی مانند چک‌باکس‌ها و اینپوت‌ها استفاده کنید تا از شخصی خواسته شود اطلاعات خاصی را وارد کند، همچنین با متنی ساده برای دسته‌بندی اطلاعات نیز قابل استفاده می‌باشد.

۱. سایزها — Regular

سه سایز با وزن فونت Regular (400)

label--large
label--medium
label--small

۲. سایزها — Semibold

سه سایز با وزن فونت Semibold (500)

label--large label--semibold
label--medium label--semibold
label--small label--semibold

۳. Required — Regular

نمایش ستاره اجباری (*) با سایزهای مختلف

large + required
medium + required
small + required

۴. Required — Semibold

نمایش ستاره اجباری (*) با وزن فونت Semibold

large + semibold + required
medium + semibold + required
small + semibold + required

۵. Disabled

حالت غیرفعال — رنگ متن و ستاره هر دو #bbbcbe

Regular — بدون Required

large + disabled
medium + disabled
small + disabled

Regular — با Required

large + required + disabled
medium + required + disabled
small + required + disabled

Semibold — با Required

large + semibold + required + disabled
medium + semibold + required + disabled
small + semibold + required + disabled

۹. CSS Classes

کلاس توضیح
labelکلاس پایه (روی <label>)
label--largeسایز بزرگ (16px Regular / 16px Semibold)
label--mediumسایز متوسط (14px Regular / 12px Semibold)
label--smallسایز کوچک (12px)
label--semiboldوزن فونت Semibold (500)
label--requiredنشانگر فیلد اجباری (semantic)
label--disabledحالت غیرفعال
label__textمتن برچسب
label__requiredستاره اجباری (*)

۱۰. نحوه استفاده

ساده (بدون Required)

<label class="label label--medium">
  <span class="label__text">برچسب</span>
</label>

با Required

<label class="label label--medium label--required">
  <span class="label__text">برچسب</span>
  <span class="label__required">*</span>
</label>

Semibold + Required

<label class="label label--large label--semibold label--required">
  <span class="label__text">برچسب</span>
  <span class="label__required">*</span>
</label>

Disabled + Required

<label class="label label--medium label--required label--disabled">
  <span class="label__text">برچسب</span>
  <span class="label__required">*</span>
</label>

مرتبط کردن با فیلد (for/id)

<label class="label label--medium label--required" for="email-input">
  <span class="label__text">ایمیل</span>
  <span class="label__required">*</span>
</label>
<input id="email-input" type="email" class="input__field" />
