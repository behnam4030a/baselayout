Button Component

کامپوننت دکمه - Pure CSS Component

درباره این کامپوننت:

  • 6 نوع دکمه: Primary, AI, Secondary, Subtle, Outline, Transparent
  • 3 سایز: Large (45px), Medium (37px), Small (24px)
  • 4 حالت: Rest, Hover, Pressed, Selected
  • پشتیبانی کامل از RTL/LTR
  • بدون نیاز به JavaScript - فقط CSS classes

نحوه استفاده

استفاده ساده (توصیه می‌شود)

<!DOCTYPE html> <html lang="fa" dir="rtl"> <head> <link rel="stylesheet" href="button.css"> </head> <body> <!-- فقط HTML + CSS classes - بدون JavaScript! --> <button class="btn btn--primary btn--large"> ذخیره </button> </body> </html>

گزینه‌های موجود (CSS Classes)

<!-- Type (نوع) --> btn--primary | btn--ai | btn--secondary | btn--subtle | btn--outline | btn--transparent <!-- Size (سایز) --> btn--small | btn--medium | btn--large <!-- State (حالت) --> btn--disabled (+ disabled attribute) btn--selected (+ aria-pressed="true")
