Checkbox Component

کامپوننت چک‌باکس - Pure CSS Component

درباره این کامپوننت:

  • 3 وضعیت منطقی (Status): Unchecked, Checked, Indeterminate
  • 5 حالت تعاملی (State): Rest, Hover, Pressed, Focused, Disabled
  • 2 سایز: Large (28px), Medium (24px)
  • Label اختیاری با قابلیت کلیک
  • پشتیبانی کامل از RTL/LTR
  • Accessible: WCAG 2.1 Level AA

نحوه استفاده

استفاده ساده

<link rel="stylesheet" href="checkbox.css"> <script src="checkbox.js" defer></script> <!-- Checkbox با برچسب --> <label class="checkbox checkbox--large"> <input type="checkbox" class="checkbox__input" /> <span class="checkbox__label">برچسب</span> <span class="checkbox__box"> <span class="checkbox__inner"> <svg class="checkbox__icon checkbox__icon--check" ...>...</svg> <svg class="checkbox__icon checkbox__icon--indeterminate" ...>...</svg> </span> </span> </label>

Checked (انتخاب شده)

<label class="checkbox checkbox--large"> <input type="checkbox" class="checkbox__input" checked /> ... </label>

Disabled (غیرفعال)

<label class="checkbox checkbox--large checkbox--disabled"> <input type="checkbox" class="checkbox__input" disabled /> ... </label>

Indeterminate (نامعین)

<!-- فقط data-indeterminate را روی input قرار دهید --> <label class="checkbox checkbox--large"> <input type="checkbox" class="checkbox__input" data-indeterminate /> ... </label> <!-- یا به صورت دستی با JavaScript --> document.querySelector('.checkbox__input').indeterminate = true;

بدون برچسب

<label class="checkbox checkbox--large"> <input type="checkbox" class="checkbox__input" aria-label="توضیح" /> <span class="checkbox__box">...</span> </label>

Select All (انتخاب همه)

<!-- والد --> <input class="checkbox__input" data-checkbox-group="my-group" data-checkbox-role="parent" /> <!-- فرزندان --> <input class="checkbox__input" data-checkbox-group="my-group" data-checkbox-role="child" />
