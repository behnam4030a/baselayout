Tab Component

تب (Tab) یعنی سوئیچ بین چند نمای هم‌سطح داخل یک صفحه — بدون اینکه کاربر از صفحه خارج شود.

۱. تب خط زیر (Underline — پیش‌فرض)

تب فعال با خط زیر سبز و فونت Medium مشخص می‌شود. Container پس‌زمینه سفید دارد.

محتوای تب «جزئیات حساب» — اطلاعات حساب کاربری شامل نام، ایمیل و شماره تماس.
محتوای تب «اطلاع رسانی» — تنظیمات ارسال اعلان‌ها از طریق ایمیل و پیامک.
محتوای تب «امنیت» — تغییر رمز عبور و فعال‌سازی احراز هویت دو مرحله‌ای.

۲. تب پر شده (Filled / Pill)

تب فعال با پس‌زمینه سبز و متن سفید مشخص می‌شود. Container بدون پس‌زمینه سفید است.

محتوای تب «جزئیات حساب» — نمایش در حالت Filled.
محتوای تب «اطلاع رسانی» — نمایش در حالت Filled.
محتوای تب «امنیت» — نمایش در حالت Filled.

۳. تب رنگی (Colored — متن فعال brand)

تب فعال با متن سبز برند نمایش داده می‌شود. با افزودن کلاس tab--colored فعال می‌شود.

محتوای تب «جزئیات حساب»
محتوای تب «اطلاع رسانی»
محتوای تب «امنیت»

۴. تب با آیکن (Underline + Icon)

هر تب دارای آیکن SVG در کنار متن است. Container وسط‌چین می‌شود.

محتوای تب «جزئیات حساب» — نمایش با آیکن.
محتوای تب «اطلاع رسانی» — نمایش با آیکن.
محتوای تب «امنیت» — نمایش با آیکن.

۵. تب با Badge (Underline + Badge)

تب فعال دارای عدد/badge در کنار متن است.

محتوای تب «جزئیات حساب».
محتوای تب «اطلاع رسانی».
محتوای تب «امنیت» — ۳ هشدار امنیتی فعال وجود دارد.

۶. تست API جاوااسکریپت
