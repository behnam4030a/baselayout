Input Component

کامپوننت اینپوت - Pure CSS Component

درباره این کامپوننت:

اینپوت‌ها به کاربران اجازه می‌دهند تا داده‌های متنی کوتاه و آزاد را وارد کنند. اگر نیاز به جمع‌آوری بیش از یک خط اطلاعات دارید، Textarea را امتحان کنید.

  • 6 وضعیت تعاملی (State): Rest, Hover, Focused, Error, Disabled, ReadOnly
  • 3 سایز: Large (45px), Medium (37px), Small (24px)
  • آیکن قبل/بعد از متن (اختیاری)
  • Divider عمودی بین آیکن before و متن
  • Accessible Indicator (خط پایینی با box-shadow)
  • پشتیبانی کامل از RTL/LTR
  • Accessible: WCAG 2.1 Level AA

نحوه استفاده

ساده‌ترین حالت (بدون آیکن)

<link rel="stylesheet" href="input.css"> <div class="input input--large"> <div class="input__container"> <input type="text" class="input__field" placeholder="متن نگهدارنده" /> </div> </div>

با آیکن Before و Divider

<div class="input input--large"> <div class="input__container"> <span class="input__icon input__icon--before"> <svg>...</svg> </span> <span class="input__divider"></span> <input type="text" class="input__field" placeholder="متن نگهدارنده" /> </div> </div>

با هر دو آیکن

<div class="input input--medium"> <div class="input__container"> <span class="input__icon input__icon--before"><svg>...</svg></span> <span class="input__divider"></span> <input type="text" class="input__field" placeholder="متن نگهدارنده" /> <span class="input__icon input__icon--after"><svg>...</svg></span> </div> </div>

Error State

<!-- کلاس input--error را روی wrapper اضافه کنید --> <div class="input input--large input--error"> <div class="input__container"> <input type="text" class="input__field" placeholder="متن نگهدارنده" aria-invalid="true" /> </div> </div> <!-- Set/Remove error با JavaScript --> document.querySelector('.input').classList.add('input--error'); document.querySelector('.input').classList.remove('input--error');

Disabled

<div class="input input--large"> <div class="input__container"> <input type="text" class="input__field" placeholder="متن نگهدارنده" disabled /> </div> </div>

ReadOnly

<div class="input input--large"> <div class="input__container"> <input type="text" class="input__field" value="متن فقط‌خواندنی" readonly /> </div> </div>

CSS Classes

Class توضیح
input کلاس پایه (روی wrapper div)
input--large سایز بزرگ (45px)
input--medium سایز متوسط (37px)
input--small سایز کوچک (24px)
input--error حالت خطا
input--disabled حالت غیرفعال (fallback، ترجیحاً disabled attr روی input)
input--readonly حالت فقط‌خواندنی (fallback، ترجیحاً readonly attr روی input)
input--full-width عرض کامل (100%)
input__container کانتینر داخلی (flex)
input__field فیلد ورودی بومی (input)
input__icon آیکن wrapper
input__icon--before آیکن قبل از متن
input__icon--after آیکن بعد از متن
input__divider خط جداکننده عمودی (بین آیکن before و متن)

اولویت State ها

اولویت State نحوه فعال‌سازی
1 (بالاترین) Disabled disabled attribute روی input
2 Error input--error class روی wrapper
3 Focused فوکوس روی فیلد (خودکار با :focus-within)
4 Hover ماوس روی input (خودکار با :hover)
5 (پایین‌ترین) Rest حالت پیش‌فرض
